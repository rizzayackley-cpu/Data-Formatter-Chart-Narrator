## Contributing

Thanks for your interest in contributing!

### Getting Started
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build: `npm run build`

### Guidelines
- Keep changes incremental and focused.
- Reuse existing patterns in `src/app/domain` and `src/app/pages/workbench`.
- Avoid committing generated outputs or secrets (see `.gitignore`).

### Pull Requests
- Describe what changed and why.
- Include verification steps (screenshots or the exact commands you ran).
